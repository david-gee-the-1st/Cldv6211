Sabai Baskets

R300.00

The Sabai braiding which originates from eastern rural parts of West Bengal, has been used over generations and now re-modernized to create a phenomenal style of basket making. This type of braiding uses harvested, dried and dyed grass to create some of the most beautifully patterned products to use and enjoy in our daily lives. Interesting fact is that the grass used to make these products is bent by hand and smoothened using rubber tree trunks.
